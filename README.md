A real-time maritime monitoring system that tracks global port activities, monitors ship movements, and provides operational analytics.

🚀 Overview
The HH Port Analytics Dashboard provides a centralized platform to monitor port locations across the globe. It leverages a live map interface for spatial tracking and an analytics engine to provide insights into port operations.

🛠 Features
Interactive Map: Built with Leaflet.js to visualize global port locations.

Real-Time Simulation: Simulated maritime traffic visualization showing active ship movements between ports.

Operational Analytics: Sidebar navigation and data-driven insights for each tracked port.

Live Dashboard: Real-time clock and status indicators for system monitoring.

⚙️ Tech Stack
Frontend: HTML5, CSS3, JavaScript (Leaflet.js)

Backend: Node.js, Express.js

Database: PostgreSQL (via Supabase)

Deployment: Render

📋 Installation & Setup
Prerequisites
Node.js installed

Supabase account with a configured database
